# 🚀 MyProjects

A collection of my interactive web projects built using HTML, CSS, and JavaScript.

---

## 🌟 Features

- 🌙 Dark / Light Theme Toggle  
- 🎯 Action-based Buttons  
  - Single Click → ToDo List  
  - Double Click → PDF Merger  
  - Hover → Custom Links  
- 🔗 Dynamic Link System  
  - Add any website / GitHub profile  
  - Auto username extraction  
  - Choose interaction type (click / dblclick / hover)  
- 🧠 Local Storage Support (data persists after refresh)  
- 👉 Slide-to-open interaction (iPhone-style slider)  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 💡 How to Use

1. Paste a link in the input field  
2. Select action type (click / double click / hover)  
3. Click **Add**  
4. Interact with buttons to open links
